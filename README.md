# Windows11_and_M365_DeploymentLab

Windows 11 and Office 365 Deployment Lab

The lab contains evaluation versions of the following products:

- Windows 11 Enterprise, Version 24H2
- Microsoft Configuration Manager, Version 2409
- Windows Server 2022

The lab is also designed to be connected to trials for:

- Microsoft 365 E5
- Microsoft 365 Apps for enterprise
- Office 365 E5 with Enterprise Mobility + Security (EMS)
